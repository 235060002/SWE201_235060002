# SWE201_235060002

This project is a C# Console Application for the SWE201 lab course.

## Included Files
- `Program.cs`: Contains the Main method.
- `Car.cs`: Contains the Car class with:
  - Two member variables
  - Properties
  - An empty and a parameterized constructor
  - A member function (GetCarAge)
  - A ToString override

## Purpose
The project demonstrates:
- Creating classes in separate files
- Using constructors
- Using properties to access private fields
- Creating objects in Main
- Performing operations and printing results
